# 線上點餐系統

這是一項基於 Vue3 + TypeScript + Vite 所搭建之線上點餐系統，其中包含了：

1. 會員系統
    - 新增會員
    - 會員登入
    - 修改會員資料
2. 訂餐系統
   - 新增訂單
   - 查詢訂單
   - 刪除訂單
3. 全站RWD實作
